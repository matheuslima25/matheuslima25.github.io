Branding

Branding ou brand management (gestão de marca) refere-se à gestão da marca de uma empresa, tais como seu nome, as imagens ou ideias a ela associadas, incluindo slogans, símbolos, logotipos e outros elementos de identidade visual que a representam ou aos seus produtos e serviços

Ilustrações

Um ilustrador é um artista especializado em melhorar a escrita ou a elucidação de conceitos, fornecendo uma representação visual que corresponde ao conteúdo do texto ou ideia associada.

Web Design

O web design é uma extensão da prática do design gráfico, onde o foco do projeto é a criação de web sites e documentos disponíveis no ambiente da World Wide Web (o famoso "WWW").

Web Development

Web Development (desenvolvimento web) é o termo utilizado para descrever o desenvolvimento de sites, na Internet. Este é o profissional que trabalha desenvolvendo websites, podendo ser um Web Designer (Desenvolvedor do Layout), ou Web Developer(Desenvolvedor de sistemas).
